# Smart Attendance System

AI based attendance system using:

• QR Code Verification  
• GPS Location Verification  
• Face Recognition  
• Admin Analytics Dashboard  
• Excel Attendance Export  

## Technologies Used

Python Flask  
MySQL  
OpenCV  
Face Recognition  
HTML CSS JavaScript  

## Features

Student Registration  
Face Registration  
QR Attendance  
GPS Verification  
Admin Dashboard  
Attendance Analytics
